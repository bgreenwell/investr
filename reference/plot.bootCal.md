# Plots method for bootstrap calibration

The [`plot()`](https://rdrr.io/r/graphics/plot.default.html) method for
`"bootCal"` objects. In particular, this method takes a `"bootCal"`
object and produces plots for the bootstrap replicates of the inverse
estimate.

## Usage

``` r
# S3 method for class 'bootCal'
plot(x, ...)
```

## Arguments

- x:

  An object that inherits from class `"bootCal"`.

- ...:

  Additional optional arguments. At present, no optional arguments are
  used.

## Value

`x` is returned invisibly.
