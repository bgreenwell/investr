# Package index

## Inverse estimation

Point and interval estimates for the unknown predictor value
corresponding to an observed (or specified mean) value of the response.

- [`invest()`](https://bgreenwell.github.io/investr/reference/invest.md)
  : Inverse estimation for linear, nonlinear, and generalized linear
  models
- [`invest(`*`<survfit>`*`)`](https://bgreenwell.github.io/investr/reference/invest.survfit.md)
  [`invest(`*`<Surv>`*`)`](https://bgreenwell.github.io/investr/reference/invest.survfit.md)
  : Inverse estimation for Kaplan-Meier survival curves
- [`calibrate()`](https://bgreenwell.github.io/investr/reference/calibrate.md)
  : Calibration for the simple linear regression model

## Prediction and plotting

Predictions, standard errors, and confidence/prediction bands for fitted
regression models.

- [`predFit()`](https://bgreenwell.github.io/investr/reference/predFit.md)
  : Predictions from a Fitted Model
- [`plotFit()`](https://bgreenwell.github.io/investr/reference/plotFit.md)
  : Plotting fitted models
- [`plot(`*`<bootCal>`*`)`](https://bgreenwell.github.io/investr/reference/plot.bootCal.md)
  : Plots method for bootstrap calibration
